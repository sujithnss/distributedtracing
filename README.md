# DistributedTracing

To Run Do the following

1)       Do Vagrant up

2)       Access the demo API over http://10.0.0.30:8081/

3)       Access the Zipkins UI over http://10.0.0.30:9411/

Pump some data using WRK or CURL and see the Trace output.
